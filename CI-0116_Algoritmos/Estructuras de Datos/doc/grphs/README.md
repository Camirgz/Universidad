### Comparación de estructuras de datos

Este notebook genera gráficos que comparan diferentes estructuras de datos y muestra visualmente los tiempos de ejecución para distintos métodos de búsqueda y eliminación de datos.

---

### Requisitos

* **Python 3.6 o superior**
* **pip** (gestor de paquetes de Python)
* Las siguientes bibliotecas de Python:

  * `matplotlib`
  * `numpy`
  * `notebook` (para ejecutar Jupyter Notebooks)

---

### Instalación en Linux

Sigue estos pasos desde una terminal:

1. **Actualizar paquetes del sistema (opcional pero recomendado)**

```bash
sudo apt update
```

2. **Instalar Python 3 y pip (si no los tienes ya)**

```bash
sudo apt install python3 python3-pip -y
```

3. **Instalar dependencias del proyecto (incluyendo Jupyter Notebook)**

```bash
pip3 install matplotlib numpy notebook
```

---

### Cómo ejecutar el notebook

Desde Visual Studio Code simplemente de click en ejecutar todas las celdas y los grádicos serán mostrados

**Desde la terminal de Linux**

1. Después de clonar el repositorio, dirígete al directorio:

```bash
cd Tareas_Programadas/tp/doc/grphs
```

2. Inicia Jupyter Notebook:

```bash
jupyter notebook
```

3. Se abrirá una ventana en tu navegador. Haz clic en `grphs.ipynb` para abrir y ejecutar el notebook.


4. Si esto no funciona, ejecute 

```bash
python3 -m notebook
```

y siga las instrucciones mostradas en la línea de comandos
