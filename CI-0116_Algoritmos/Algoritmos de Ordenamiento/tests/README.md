# Pruebas.sh

Script para compilar y ejecutar el programa automáticamente.

## ¿Qué hace?

- Compila los archivos de `source/` (`main.cpp`, `ControladorArrays.cpp`, `Ordenador.cpp`).
- Ejecuta el programa 3 veces.
- Guarda las salidas en `tests/out1.txt`, `out2.txt` y `out3.txt`.

## Uso

Desde la carpeta `tests/`:

```bash
chmod +x Pruebas.sh
./Pruebas.sh
```

## Requisitos

- Tener `g++` instalado.
- Mantener la estructura de carpetas original.
